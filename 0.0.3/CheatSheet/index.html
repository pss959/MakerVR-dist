MakerVR Cheat Sheet

Category Name Radial Menu Icon Description Keyboard Shortcut Controller Shortcut
Basics Undo Undo the last command <Ctrl>Z
Basics Redo Redo the last undone command <Ctrl>Y
Basics OpenHelpPanel Open the panel to show help F1, <Shift>Slash
Basics Quit Exit the application <Ctrl>Q
Creation CreateBox Create a primitive Box object
Creation CreateCylinder Create a primitive Cylinder object
Creation CreateSphere Create a primitive Sphere object
Creation CreateTorus Create a primitive Torus object
Creation ImportModel Import or reimport a model from a file <Ctrl>I
Combining CSGDifference Create a CSG Difference or change selected models to a CSG Difference Minus
Combining CSGIntersection Create a CSG Intersection or change selected models to a CSG Intersection <Shift>Alpha7
Combining CSGUnion Create a CSG Union or change selected models to a CSG Union <Shift>Backslash
Combining Hull Create a model that is the convex hull of selected models
Editing Copy Copy all selected models to the clipboard <Ctrl>C
Editing Cut Cut all selected models to the clipboard <Ctrl>X
Editing Delete Delete all selected models Backspace
Editing Paste Paste all models from the clipboard <Ctrl>V
Editing PasteInto Paste all models from the clipboard as children of the selected model <Shift><Ctrl>V
File OpenSessionPanel Open the panel to save or open session files <Ctrl>S
File OpenSettingsPanel Edit application settings <Ctrl>P
Layout TogglePointTarget TOGGLE: Activate or deactivate the point target
Layout ToggleEdgeTarget TOGGLE: Activate or deactivate the edge target
Layout LinearLayout Lay out the centers of the selected models along a line using the edge target
Layout RadialLayout Lay out selected models along a circular arc
Layout ToggleAxisAligned TOGGLE: Transform models in local or global coordinates
Modification DecreaseComplexity Decrease the complexity of the selected models by .05 <Shift>Comma
Modification EditName Edit the name of the primary selection <Ctrl>N
Modification IncreaseComplexity Increase the complexity of the selected models by .05 <Shift>Period
Modification MoveNext Move the selected model down in the order
Modification MovePrevious Move the selected model up in the order
Modification MoveToOrigin Move the primary selection to the origin Equals
None None
Selection SelectAll Select all top-level models <Ctrl>A
Selection SelectFirstChild Select the first child of the primary selection <Ctrl>DownArrow
Selection SelectNextSibling Select the next sibling of the primary selection <Ctrl>RightArrow
Selection SelectNone Deselect all selected models <Shift><Ctrl>A
Selection SelectParent Select the parent of the primary selection <Ctrl>UpArrow
Selection SelectPreviousSibling Select the previous sibling of the primary selection <Ctrl>LeftArrow
State DecreasePrecision Decrease the current precision Z Down
State IncreasePrecision Increase the current precision X Up
Tools1 MirrorTool Mirror selected models in any dimension (Alt for in-place)
Tools1 RotationTool Rotate the selected models (Alt for in-place)
Tools1 ScaleTool Change the size of the selected models (Alt for symmetric)
Tools1 SwitchToNextTool Activate the enabled tool to the right of the current one RightBracket Right
Tools1 SwitchToPreviousTool Activate the enabled tool to the left of the current one LeftBracket Left
Tools1 TranslationTool Change the position of the selected models
Tools2 ColorTool Edit the color of the selected models
Tools2 ComplexityTool Edit the complexity of the selected models
Tools2 CylinderTool Change the top and bottom radii of the selected Cylinder
Tools2 ProfileTool Modify the edge profiles of selected models
Tools2 TorusTool Change the inner and outer radii of the selected Torus
Viewing HideSelected Hide selected top-level models <Ctrl>H
Viewing InspectSelection Inspect the current primary selection <Ctrl>T
Viewing ShowAll Show all hidden top-level models <Shift><Ctrl>H
Viewing ToggleShowEdges TOGGLE: Show or hide edges on all models <Ctrl>E
